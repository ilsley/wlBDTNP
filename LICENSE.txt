The scripts in this project convert data from the Berkeley Drosophila Transcription Network Project (BDTNP) to Mathematica format. 
The original data were downloaded from http://bdtnp.lbl.gov/Fly-Net/bioimaging.jsp?w=releases. 
If you make use of the data, please cite this original source as appropriate, for example: Fowlkes et al. (2008). A Quantitative Spatiotemporal Atlas of Gene Expression in the Drosophila Blastoderm. Cell 133, 364â€“374.
